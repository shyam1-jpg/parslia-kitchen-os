WHERE IS THE START FILE?
========================

Windows often HIDES the ".bat" ending.

So the file may look like this in the folder:

   START
   DOUBLE-CLICK-ME
   1-START-HERE
   CHECK
   TEST-NOW

Those ARE the starter files.

What to do:
1. Make sure you are inside the folder named:  pc-guard
   (not only the outer zip folder)

2. Double-click one of these:
   - DOUBLE-CLICK-ME
   - 1-START-HERE
   - START

3. Keep the black window open.

4. If a browser page does not open, double-click:
   OPEN-DASHBOARD

To show the full names in Windows:
- Open File Explorer
- Click View
- Tick "File name extensions"
Then you will see START.bat
