PC Guard — how to run (simple)

IMPORTANT: This version does NOT need Windows Firewall permission.
It opens a normal file page (data\live.html), not a website port.

1) Download this zip and extract it:
   https://github.com/shyam1-jpg/parslia-kitchen-os/archive/refs/heads/cursor/pc-usage-guard-d44d.zip

2) Open the extracted folder, then open:
   pc-guard

3) Double-click: START.bat
   - Keep the black window open
   - A browser page should open automatically (live.html)

4) You should see a file event like:
   PC_GUARD_STARTED_....txt

5) To test again, double-click: TEST-NOW.bat
   Then look at the dashboard (it refreshes every 3 seconds)

If the browser did not open:
- Double-click OPEN-DASHBOARD.bat
- Or open this file manually:
  pc-guard\data\live.html

Firewall note:
- Normal START.bat does NOT need firewall allow
- Only START-WEB.bat uses a port (optional). If you use that and
  Windows blocks it, run FIX-FIREWALL.bat as Administrator.

If START.bat fails:
- Install Python from https://www.python.org/downloads/
- Tick "Add python.exe to PATH"
- Run CHECK.bat and send a screenshot of the black window
