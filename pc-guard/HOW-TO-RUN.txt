PC Guard — how to run (simple)

IMPORTANT:
Windows may hide ".bat", so START.bat can look like just "START".
Use DOUBLE-CLICK-ME if that is easier to find.

1) Download this zip and extract it:
   https://github.com/shyam1-jpg/parslia-kitchen-os/archive/refs/heads/cursor/pc-usage-guard-d44d.zip

2) Open the extracted folder, then open THIS folder:
   pc-guard

   You should see files like:
   DOUBLE-CLICK-ME
   START
   CHECK
   TEST-NOW
   READ-ME-FIRST

3) Double-click: DOUBLE-CLICK-ME
   (or START / 1-START-HERE)
   - Keep the black window open
   - A browser page should open (live.html)

4) To test, double-click: TEST-NOW

If the browser did not open:
- Double-click OPEN-DASHBOARD
- Or open: pc-guard\data\live.html

To see .bat endings in Windows:
File Explorer -> View -> tick "File name extensions"
